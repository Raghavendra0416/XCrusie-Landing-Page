Project Name: xCruise Landing Page
Description: xCruise is a fully responsive travel landing page designed to showcase exotic cruise destinations and vacation deals. The project is built using semantic HTML5 and CSS3, focusing on precise layout control and visual hierarchy. It features a dynamic grid layout for travel packages that adapts seamlessly across mobile, tablet, and desktop devices.

Key Features:
Responsive Navigation: A distinct navy-blue navigation bar with logo and menu links.
Immersive Hero Section: Features a full-width background image with a clear Call-to-Action ("Book Now").
Destination Showcase: A complex "Discover" section containing:
"Popular" filter pills with specific box-shadow styling.
A promotional banner overlay for Caribbean cruises.
A responsive card grid displaying 8 different destinations (adjusting from 1 to 4 columns based on screen width).
Contact & Footer: structured contact information including operation hours and social media integration.

Tech Stack:
HTML5 (Semantic Tags: <nav>, <section>, <header>, <footer>)
CSS3 (Flexbox, CSS Grid, Media Queries)
Responsive Design Breakpoints: Mobile (<480px), Tablet (480-768px), Desktop (>768px).
